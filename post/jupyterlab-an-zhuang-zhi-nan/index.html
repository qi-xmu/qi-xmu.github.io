Jupyterlab安装指南

2021-03-07 3 min read # jupyterlab # C++

简单整理了一下window安装cling环境的步骤。

更新 2021-03-07

  • win家庭版也可安装wsl，这里建议使用wsl2。其实Hyper-V是一个很好用的虚拟化平台，可以创建虚拟机。详情点击

安装wsl

只有window专业版才能使用wsl功能，TB购买就可以，可以在控制面板-程序中打开。

选择启用或关闭Window功能

然后在微软商店安装：

!

启动设置用户名和密码。

打开后，输入

sudo -i

输入密码（不可见）

安装anaconda

下载文件

# 这里需要打开文件所在路径，下图可以看到C盘，D盘在linux中挂载的位置，打开你下载文件的位置
cd [file path]

可以看到 anaconda文件，然后输入

# tab可以自动补全文件名
bash Anaconda3-2020.11-Linux-x86_64.sh

后面一直ENTER，需要输入的地方都输入yes。等待安装完成。

激活环境

cd ~
source .bashrc

安装jupyterlab3.0

创建环境

conda create --name cling python=3.8

激活环境

conda activate cling

安装jupyterlab和xeus-cling

python -m pip install --upgrade pip
pip install jupyterlab
conda install -c conda-forge xeus-cling

如果网速不行可以换源清华源

conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free/
conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud/conda-forge 
conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud/msys2/
# 设置搜索时显示通道地址
conda config --set show_channel_urls yes
# 显示检索路径，每次安装包时会将包源路径显示出来
conda config --set show_channel_urls yes
conda config --set always_yes True
# 显示所有镜像通道路径命令
conda config --show channel

### pip换源
mkdir ~/.pip
nano ~/.pip/pip.conf
## 输入
[global]
index-url = https://mirrors.aliyun.com/pypi/simple/
[install]
trusted-host = mirrors.aliyun.com

# 换回默认源
conda config --remove-key channels

安装完成之后可以配置jupyterlab，也可能不需要配置。如果输入jupyter lab --allow-root可以正常启动就不用配置吧。

jupyter lab --generate-config

这个命令会在~/.jupyter/目录下生成配置文件。可以用nano进行编辑

nano `~/.jupyter/jupyter_notebook_config.py

可以输入，如果需要设置密码建议bing一下。这个地方有很奇怪的地方。

c.NotebookApp.allow_remote_access = True
c.NotebookApp.open_browser = False
c.NotebookApp.ip='*'
c.NotebookApp.allow_root = True
c.NotebookApp.notebook_dir = '/mnt/c/Users/36014/项目' #初始路径
# c.NotebookApp.port = 8080 #自定义端口

安装中文插件

下载中文插件，打开文件对应路径，执行

pip install jupyterlab_language_pack_zh_CN-0.0.1.dev0-py2.py3-none-any.whl

选择中文

选择C++标准就可以编写代码了。

可能出现的问题

需要密码

在终端Ctrl + C，可以显示出token，复制token，然后设置自己的密码。
