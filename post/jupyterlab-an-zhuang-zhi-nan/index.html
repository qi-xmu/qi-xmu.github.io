Jupyterlab安装指南

2021-03-07 6 min read # jupyterlab # C++

简单整理了一下window安装cling环境的步骤。

更新 2021-03-09

  • 更新安装命令pip install jupyterlab -i https://pypi.tuna.tsinghua.edu.cn/simple以防网速问题，这里改用清华源。
  • 今天试了一下 Kali Linux 这个系统，也挺好用的，大家可以试试。

更新 2021-03-07

  • 推荐微软官方终端Terminal。可以在微软商店下载。详情点击
  • win家庭版也可安装wsl，这里建议使用wsl2。安装Docker的时候需要Hyper-V才需要Window专业版，这里我弄混了。Hyper-V是一个很好用的虚拟化平台，可以创建虚拟机。详情点击

安装wsl

官网：安装方法参考链接，建议直接看官方文档。详情点击

只有window专业版才能使用wsl功能，TB购买就可以，可以在控制面板-程序中打开。

选择启用或关闭Window功能

然后在微软商店安装：

!

启动设置用户名和密码（在Linux系统中，输入密码的字符是不显示的，但不代表没有）。😇

打开后，输入

sudo -i

输入密码（不可见）

安装anaconda

官网请点击

下载文件点击下载

在Ubuntu终端中，安装anaconda。（不要直接打开下载文件）

# 这里需要打开文件所在路径，下图可以看到C盘，D盘在linux中挂载的位置，打开你下载文件的位置。
# 这个path是文件所在路径，注意虚拟中的路径和win不一样。
cd [file_path]

可以看到 anaconda文件，然后输入

# tab可以自动补全文件名
bash Anaconda3-2020.11-Linux-x86_64.sh

如果上面这一步看不懂，找不到路径，按照下面的步骤执行

  1. 将下载的文件放在C盘根目录
  2. 执行下面代码：
bash /mnt/c/Anaconda3-2020.11-Linux-x86_64.sh

后面一直按ENTER键，需要同意的地方都输入yes。等待安装完成。

激活anaconda环境

cd ~
source .bashrc

激活成功后，前面会出现(base)，代表成功

安装jupyterlab3.0

注意版本号，这里强烈推荐使用Lab3.0，会比notebook好用很多

创建anaconda虚拟环境

这里如果下载缓慢，看后面conda换源，改用清华源。（Ctrl+C终止任务，需要多按几次）

conda create --name cling python=3.8

激活cling环境

conda activate cling

安装jupyterlab和xeus-cling

python -m pip install --upgrade pip
pip install jupyterlab -i https://pypi.tuna.tsinghua.edu.cn/simple
conda install -c conda-forge xeus-cling

如果网速不行可以换源清华源

nano .condarc
# 输入
channels:
  - defaults
show_channel_urls: true
default_channels:
  - https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/main
  - https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/r
  - https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/msys2
custom_channels:
  conda-forge: https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud
  msys2: https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud
  bioconda: https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud
  menpo: https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud
  pytorch: https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud
  simpleitk: https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud

按Ctrl+S保存，Ctrl+X退出

这部分命令不需要执行，看看就行~~。换源后继续执行前面的代码就可以了。

# 设置搜索时显示通道地址
conda config --set show_channel_urls yes
# 显示检索路径，每次安装包时会将包源路径显示出来
conda config --set show_channel_urls yes
conda config --set always_yes True
# 显示所有镜像通道路径命令
conda config --show channel

当你使用 pip 下载很慢的时候，可以尝试换源。

### pip换源，在用户目录下创建 .pip 目录，新建文件。
mkdir ~/.pip
nano ~/.pip/pip.conf
## 输入
[global]
timeout =6000
index-url =http://pypi.douban.com/simple/
[install]
use-mirrors = true
mirrors = http://pypi.douban.com/simple/
trusted-host = pypi.douban.com

# 换回默认源，可以不换回。。。
conda config --remove-key channels

安装完成之后可以配置jupyterlab，也可能不需要配置。如果输入jupyter lab --allow-root可以正常启动，就不用配置吧。

执行完jupyter lab --allow-root后，如果终端出现链接，按住Ctrl点击链接就可以打开。如果终端没出现链接，打开浏览器，输入``127.0.0.1:8888`，这个地方需要输入密码或者Token。请直接跳到文末。

安装完成

下次如何打开?
打开Ubuntu终端：

sudo -i
conda activate cling
jupyter lab --allow-root

如何关闭？
按两次Ctrl+C，或者直接关闭终端。

配置部分（仅供参考，建议直接百度）

jupyter lab --generate-config

这个命令会在~/.jupyter/目录下生成配置文件。可以用nano进行编辑

nano `~/.jupyter/jupyter_notebook_config.py

可以输入，如果需要设置密码建议bing一下。这个地方有很奇怪的地方。

c.NotebookApp.allow_remote_access = True
c.NotebookApp.open_browser = False
c.NotebookApp.ip='*'
c.NotebookApp.allow_root = True
c.NotebookApp.notebook_dir = '/mnt/c/Users/36014/项目' #初始路径
# c.NotebookApp.port = 8080 #自定义端口

安装中文插件

下载中文插件，打开文件对应路径，执行

pip install jupyterlab_language_pack_zh_CN-0.0.1.dev0-py2.py3-none-any.whl

上面不懂的，参照安装anaconda的过程

选择中文

选择C++标准就可以编写代码了。
