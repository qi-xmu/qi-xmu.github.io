Ptyon: CSV文件操作

2021-03-08 3 min read

利用python对csv文件进行操作的库主要有两种：csv 和 pandas。推荐使用pandas库，处理数据更加高效。

利用CSV库

按照列表的方式进行读写
# 读取csv文件
import csv
with open('some.csv', 'r') as f:
    reader = csv.reader(f)
    for row in reader:
    	print(row)
### 说明：
### 这里返回的 row 是一个列表，包含一行中所有元素。

# 写入csv文件
import csv
with open('some.csv', 'w', newline = "") as f:
    writer = csv.writer(f, delimiter=', ')	# delimiter 分隔符
    writer.writerows(list1)
### 说明：
### 写入时和读取时对应，传入的应该是一个列表。如果是写入字符串，记得加上[]。
### newline = "" 是为了防止出现对的空行。
按照字典的方式进行读写
### 定义
class csv.DictReader(csvfile, fieldnames=None, restkey=None, restval=None, dialect='excel', *args, **kwds)

class csv.DictWriter(csvfile, fieldnames, restval='', extrasaction='raise', dialect='excel', *args, **kwds)

# 读取csv文件
with open('names.csv') as csvfile:
	reader = csv.DictReader(csvfile)
	for row in reader:
	print(row['header_name'])
### 说明：
### 这里返回的 row 是一个列表，key = header_name。

# 写入csv文件
import csv
with open('names.csv', 'w', newline='') as csvfile:
    writer = csv.DictWriter(csvfile, fieldnames="fieldnames")
    writer.writeheader()
    writer.writerow(dict1)
### 说明：
### 这里 dict1 的 key 必须和 fieldnames 相对应，不然会造成错误。
