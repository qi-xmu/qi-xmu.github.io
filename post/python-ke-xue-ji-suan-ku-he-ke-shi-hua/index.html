Python 科学计算库和可视化

2021-03-10 5 min read # jupyterlab # python

利用Python处理数据，一般常用的库包括 Numpy，Pandas，Matplotlib，PIL。下面是他们的基本使用方法。

Numpy库

**Numpy **是Python科学计算库的基础。包含了强大的N维数组对象和向量运算。

中文文档 | 官方文档

安装：pip install numpy

import numpy as np

array 的生成

arr = np.array(list1/tuple1)
# eg.
arr = np.array([[1,2,3],
                 [4,5,6]])
# range
array = np.arange(start, end, step)
# eg.

array = np.arange(10, 31, 5)	# [10 15 20 25 30]
# one
arr2 = np.ones([2,3],dtype=np.int64) 	# [[1 1 1]
 										# [1 1 1]]

数组的方法

方法 解释
arr.shape 数组的形状。返回 (row, col)
array.ndim 数组的维度。2维数组返回 2
array.size 返回 数组元素的个数。
array.dtype 返回 数组元素类型。
array.reshape(row, col) 重新定义数组的形状。

数组的计算

大小相等的数组之间的任何算术运算都会将运算应用到元素级。同样，数组与标量的算术运算也会将那个标量值传播到各个元素。

np.dot(arr1,arr2) # 返回矩阵相乘结果
np.sum(arr3,axis=0)

说明：axis=1 每一行求和；axie=0 每一列求和。若未定义axis，会把整个数组看成一维。

方法 解释
np.dot(arr1, arr2) 矩阵相乘
np.sum(arr, axis) 矩阵求和。
np.max(arr, axis) 求数组arr最大值。
np.min(arr, axis) 求数组arr的最小值。
np.mean(arr, axis) 求数组的平均值。
np.argmax(arr) 求数组最大值的坐标。二维数组返回的仍是一维坐标。
np.argmin(arr) 求数组最小值的坐标。二维数组返回的仍是一维坐标。

import pandas as pd

Series

Series是一种类似于一维数组的对象，它由一维数组（各种numpy数据类型）以及一组与之相关的数据标签（即索引）组成。

Seris中可以使用index设置索引列表，与字典不同的是，Seris允许索引重复。

pd.Series(list1/dirt1, index=[])
# eg.
s = pd.Series(['a','b','c','d','e'])
# 访问
index = s.index
values = s.values

与普通numpy数组相比，可以通过索引的方式选取Series中的单个或一组值。

s[index]
# eg.
s[[400, 500]]

Series中最重要的一个功能是：它会在算术运算中自动对齐不同索引的数据

Series 和多维数组的主要区别在于， Series 之间的操作会自动基于标签对齐数据。因此，不用顾及执行计算操作的 Series 是否有相同的标签。

DataFrame

DataFrame是一个表格型的数据结构，类似于Excel或sql表。它含有一组有序的列，每列可以是不同的值类型（数值、字符串、布尔值等）

DataFrame既有行索引也有列索引，它可以被看做由Series组成的字典（共用同一个索引）

df = pd.DataFrame()
常用参数 说明
data 初始数据。
index 行标签。
column 列标签。

PIL库

PIL库是一个具有强大图像处理能力的第三方库。

图像的组成：由RGB三原色组成,RGB图像中，一种彩色由R、G、B三原色按照比例混合而成。0-255区分不同亮度的颜色。

图像的数组表示：图像是一个由像素组成的矩阵，每个元素是一个RGB值。

安装：pip install pillow
