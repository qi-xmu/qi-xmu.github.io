Python基础

2021-03-04

这里是关于python的一些基础知识：包括 math, string, list, tuple, dict 的常见用法。

python math模块

import math

print(math.ceil(4.1))  # 返回数字的上入整数
print(math.floor(4.9)) # 返回数字的下舍整数
print(math.fabs(-10))  # 返回数字的绝对值
print(math.sqrt(9))    # 返回数字的平方根
print(math.exp(1))     # 返回e的x次幂

python 随机数

import random
random.random() 	# 生成0~1的随机数
random.randint(1,20)# 生成1~20的随机整数

python 字符串操作

# 字符串切片 左闭右开
a = "string"
a[1:4]	# tri
# join(): 以字符作为分隔符，将字符串中所有的元素合并为一个新的字符串
new_str = '-'.join('Hello') # H-e-l-l-o

t2 = (1, 2) * 2			# t2 = (1, 2, 1, 2)

python 字典

# 定义
dict1 = {}
# list -> dict list 可以转成字典，但前提是列表中元素都要成对出现
dict1 = dict([(item1, item2), (item3, item4)])
# 字典里的函数 items()  keys() values()
dict1.item()	# 显示列表，生成迭代器
dict1.keys()	# 显示键名
dict1.values()	# 显示键值
# 字典查找
dict1.get(key)			# 如果能够取到值，则返回字典中的值，否则返回None
dict1.get(key, default) # 如果能够取到值，则返回字典中的值，否则返回default
# pop操作
dict1.pop()		# 删除并返回
